# 刷题批注助手

一个用于力扣等刷题网站的浏览器扩展，可以选中题目描述文字并添加批注。

## 功能特点

- 📝 选中文字后快速添加批注
- 💾 批注自动保存到本地
- 🎨 高亮显示已批注的文字
- 👁️ 点击高亮文字查看批注
- 🗑️ 可以删除不需要的批注
- 🔒 批注按页面 URL 独立存储

## 安装方法

### Chrome/Edge 浏览器

1. 下载或克隆此项目到本地
2. 打开浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 完成！现在可以在力扣网站使用了

## 使用方法

1. 打开力扣题目页面
2. 选中你想批注的文字
3. 点击弹出的"📝 添加批注"按钮
4. 在对话框中输入批注内容
5. 点击"保存"

查看批注：
- 已批注的文字会显示黄色高亮
- 点击高亮文字即可查看批注
- 可以点击"删除批注"按钮移除批注

## 文件说明

- `manifest.json` - 扩展配置文件
- `content.js` - 核心功能脚本
- `styles.css` - 样式文件
- `icon.png` - 扩展图标（需要自行添加）

## 扩展支持的网站

- leetcode.cn (力扣中国)
- leetcode.com (力扣国际)

如需支持其他刷题网站，可以在 `manifest.json` 的 `matches` 中添加对应的网址模式。

## 注意事项

- 批注数据存储在浏览器本地，不会上传到服务器
- 每个页面的批注独立存储
- 清除浏览器数据可能会导致批注丢失
